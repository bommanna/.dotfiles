Configuration files
===================


Installation
------------

On a mac:

.. code:: bash

  $ curl -L https://raw.github.com/mtth/.dotfiles/master/bin/install.sh | bash -s

Otherwise:

.. code:: bash

  $ git clone https://github.com/mtth/.dotfiles.git
  $ .dotfiles/bin/link.sh
  $ .dotfiles/bin/submodules.sh
